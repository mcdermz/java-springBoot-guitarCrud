Learn a new language: JAVA

API Endpoints:

* All routes start with "http://sean-guitar-crud-java.herokuapp.com"

## Get all guitars
`GET ~/guitars/`

## Find guitar by ID
`GET ~/guitars/{ id }`

## Find guitar by brand
`GET ~/guitars/find/fender`   * NOT case sensitive

## Add a guitar
`POST ~/guitars brand="Gibson" model="Les Paul"`

## Delete a guitar
`DELETE ~/guitars/1`
